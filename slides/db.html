Databases

A database is software that is good at storing lots of data
Database vs. Spreadsheet
  • Many people can use it
  • Fast for lots of data(1M, 1B, or 1T rows!)
  • It is always on
  • The source of truth
  • Good for complicated analysis
Databases live on computers
  • Cloud (e.g. Amazon, Digital Ocean)
  • Your Datacenter
  • Local (your computer)

SQL

SQL is the language used to extract data from databases

  • People use SQL to get data
  • Analytics
  • Operations
  • others?

Databases are organized

  • Databases have tables
  • Tables have columns
  • Columns have types (e.g. date, number, text)
